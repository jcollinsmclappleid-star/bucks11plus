W7) ADD PREMIUM METRICS: GAP VELOCITY + FORECAST STABILITY
-----------------------------------------------------------

Implement two new derived metrics stored in attempts.metrics or computed for dashboard:

1) Gap Velocity:
- For last N full diagnostics (exclude mini, exclude drills):
  gap_mid = abs(121 - mid_score)
- velocity = change in gap_mid from oldest to newest / number of weeks
- Display:
  “Gap velocity: improving by ~X marks per diagnostic”
OR
  “Gap reduced from 7 → 3 marks”

Logic:
- Use latest and previous full diagnostic attempts.
- If only 1 full attempt, show “Velocity available after next diagnostic”.

2) Forecast Stability:
- Measure range width + volatility_index across last attempts
- Stability improving if:
  range_width decreasing and volatility decreasing
- Display as simple status chip:
  “Stability: Improving / Variable / Stable”

These must be shown on /app/programme and /app/progress for programme users.
